# Phase 8.5 Failure Triage SOP

## Fast rerun commands

Run full suite:
```bash
cd legacy/v1
./ops/e2e_full.sh
```

Run a single spec:
```bash
cd legacy/v1/frontend
npx playwright test --config=playwright.config.ts playwright/tests/associate_onboarding.spec.ts
```

Run a single test by title:
```bash
cd legacy/v1/frontend
npx playwright test --config=playwright.config.ts -g "associate self-serve onboarding"
```

## Trace and artifact inspection

Open HTML report:
```bash
cd legacy/v1/frontend
npx playwright show-report playwright/test-results/html-report
```

Open a trace artifact:
```bash
cd legacy/v1/frontend
npx playwright show-trace playwright/test-results/artifacts/<test-folder>/trace.zip
```

Artifacts to collect for each failure:
- Screenshot(s) from `playwright/test-results/artifacts/...`
- `trace.zip`
- `network-console-summary` attachment (generated by global traps)
- DOM crawler report when crawler suite fails:
  - `playwright/test-results/dom-crawler-report.json`

## Failure classification

1. `Selector/UI flake`
- Symptoms: timeout waiting for element, element detached, occasional pass on retry.
- Action: stabilize locator to semantic role/text; reduce dependency on unstable CSS structure.

2. `Backend/API regression`
- Symptoms: deterministic 4xx/5xx in `network-console-summary`, server traceback, invariant break.
- Action: capture failing request/response and backend logs; open backend bug with endpoint + payload.

3. `Data/seed drift`
- Symptoms: missing baseline entities, empty tables, unexpected status transitions.
- Action: rerun seed (`scripts/seed_e2e.sh`) and verify deterministic dataset assumptions.

4. `Environment/infra issue`
- Symptoms: connection refused, compose service unhealthy, migration failure.
- Action: check `docker compose ps`, service logs, and migration state before retrying tests.

## Triage checklist before assigning bug

1. Reproduce once with single-spec rerun.
2. Confirm if retry passes or fails consistently.
3. Inspect trace timeline and console/network summary.
4. Label as flake vs product bug.
5. File issue with:
- exact test name
- failing route/API
- artifact paths
- repro command
- commit SHA
